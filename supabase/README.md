# Supabase

Folder ini disiapkan untuk file terkait Supabase, seperti:

- migration SQL
- policy RLS
- catatan setup project
- seed data lokal jika nanti diperlukan

Simpan migration di folder `supabase/migrations/`.

## File yang sudah disiapkan

- `supabase/migrations/20260611133000_create_debts.sql`

## Cara pakai cepat

Kalau pakai Supabase SQL Editor:

1. Buka project Supabase
2. Copy isi migration SQL
3. Jalankan di SQL Editor

Kalau pakai Supabase CLI nanti, file migration ini juga sudah siap dipakai.
